# 大作文指南

你好！我是你的申论大作文专职教师。很高兴能为你进行考前辅导。

申论大作文（文章写作）常被考生妖魔化，认为需要极高的文学素养，其实不然。申论作文本质上是**“带着镣铐跳舞”的官样文章**，它是一道客观性很强的主观题。阅卷人主要看的是你的**政治站位、逻辑框架和对材料的统筹能力**。

下面我将从复习规划、高分核心、立意确立、谋篇布局及具体实操指南五个方面，为你制定一份详尽的备考方案。

### 一、 学习规划：读写结合，重在“套路”

备考申论作文，不要陷入“听课陷阱”，核心原则是**“一读二写”**。

1. **“读”：提升阅读定力与找点能力**
   - 申论的核心在于读懂材料。你需要训练自己读六七千字材料不走神的能力。
   - **研读真题**：不要只看一遍，要反复读。第一遍做题，第二遍研究得分点来源，第三遍模拟命题人思路。
   - **积累素材**：不要盲目背诵整篇范文，而是要积累**万能的逻辑框架和起承转合的“套话”**（如《人民日报》评论员文章中的金句逻辑）。
2. **“写”：练字与模板内化**
   - **练字（卷面）**：这是阅卷的第一印象。不需要你是书法家，但必须做到**一笔一划、工工整整、绝不连笔**。卷面不好，内容再深刻也难拿高分。
   - **打造自己的模板**：不要只背市面上的通用模板，要学会“移花接木”。把好的开头、结尾、分论点扩充逻辑（如“一子落而满盘活”）改写成自己的专属套路，考场上直接套用，既省时间又出彩,,。

------

### 二、 获得高分的核心要义：结构与规范

阅卷时间非常短（通常几十秒），阅卷人不会细细品味你的文采，他们主要看以下几点：

1. **卷面（门面）**：字迹工整，卷面干净，这是及格线以上的保障。
2. **标题（灵魂）**：必须包含题干要求的**主题词**或**核心判分点**，观点要明确。
3. **框架（骨架）**：这是最核心的判分点。作文结构不是固定的“三策”或“三意义”，而是根据题干和材料要求的**客观结构**。例如，题干问“关系”，你就必须写关系段；题干暗示有“问题”，你就必须分析问题,。
4. **得分点（血肉）**：分论点的段首句必须精准概括，且尽量使用材料中的关键词或高度概括词,。

------

### 三、 如何准确获取总论点（立意）

立意跑偏，满盘皆输。确定总论点（立意）有一个核心公式：**立意 = 题干（核心） + 材料（参考）**。

1. **依据题干定主题**：
   - 如果题干直接给出了标题（如“以‘水为师’为题”），总论点就是解释这个标题的深层含义（水的精神）。
   - 如果题干给出一句引言（如“夜色难免黑凉，前行必有曙光”），你需要解释这句话的关键词及其逻辑关系（如问题+信念+对策）作为总论点。
2. **依据材料找角度**：
   - 当题干比较宽泛（如“自选角度”），你需要通过梳理材料的**高频词**、**核心观点**（特别是领导人讲话、政策文件）来确定立意,。
   - **注意**：如果题干主题与材料主题看似不完全一致（例如题干说“互联互通”，材料讲“运河文化”），**以题干为准**，材料只是论证题干的素材库。

------

### 四、 如何谋篇布局：拒绝万能，因题而异

千万不要用“万能三段论”去套所有题目。你需要根据题干的暗示来搭建结构。常见的有以下几种结构：

1. **对策型结构（“怎么办”）**
   - **适用**：题干明确要求解决问题，或材料全是问题。
   - **结构**：开头（引出论点）+ 分析段（问题/意义）+ 对策1 + 对策2 + 对策3 + 结尾。
   - **示例**：围绕“资源型城市转型”写文章。
2. **分析型结构（“为什么”）**
   - **适用**：题干侧重谈理解、谈看法，或者材料都在讲意义、原因。
   - **结构**：开头 + 含义解释 + 意义/原因1 + 意义/原因2 + 意义/原因3 + （简略对策）+ 结尾。
   - **示例**：“好政策不仅满足意愿，更滋养理性与德性”，重点写这三个意义。
3. **关系型结构（“A与B”）**
   - **适用**：题干出现两个关键词（如“有与无”、“快与慢”、“流动与新生”）。
   - **结构**：开头 + **含义段**（分别解释A和B）+ **关系段**（重点分析A与B的辩证关系，如互为支撑、相辅相成）+ 对策段（如何统筹两者）+ 结尾,。
   - *特别注意*：关系段是核心判分点，千万不能漏。
4. **综合式结构（杂烩式）**
   - **适用**：题干要素多，既有观点要解释，又有问题要解决。
   - **结构**：开头 + 解释含义 + 分析关系 + 分析问题 + 提出对策 + 结尾。

------

### 五、 具体可行的指南与示例

为了让你在考场上快速成文，我为你准备了一套可操作的“模块化”写作指南。

#### 1. 标题模板

- **原则**：简洁凝练，涵盖主题。
- **单标题**：
  - 《做好______这篇大文章》
  - 《谱写______新篇章》
  - 《让______大放异彩》
- **双标题（更推荐，信息量大）**：
  - 《（动词+主题）——由（材料/题干关键词）想到的》
  - 示例：**《做好行政执法这篇大文章——由“力”“理”“利”想到的》**。

#### 2. 开头模板（“苟日新”法）

- **逻辑**：引用名言 + 时代背景/现状 + 引出总论点。

- **模板**：

  > 古人云：“苟日新，日日新，又日新”。时代在发展，我们对于**[主题词]**的认识也在不断变化。从过去的**[旧观念/旧现状]**到现在的**[新观念/新挑战]**，我们越发认识到**[题干引言/核心观点]**的重要性。因此，我们应顺应时代潮流，做好**[总论点]**这篇大文章。

- **示例（以“提升想象力”为例）**：

  > 古人云：“苟日新，日日新，又日新”。随着时代发展，我们对于“想象力”的认识也在不断变化。尤其是随着“想象力经济”时代的到来，我们越发认识到提升想象力的重要性。所以我们应该因时而变，做好提升想象力这篇大文章。

#### 3. 分论点扩充模板（“一子落而满盘活”法）

- **逻辑**：提出分论点 + 引用原理/名言 + 结合材料案例 + 回扣论点。

- **模板**：

  > **[分论点段首句]**。这些年来，**[主题]**取得了显著成绩，究其原因在于立足**[分论点关键词]**这个支点，撬动了全局。万事万物，一理相通。通过**[概括材料案例]**这些经验，找到了做好**[分论点]**这一突破口。我国自古就有“一子落而满盘活”的哲学道理。**[分论点]**就是那个关键的“一子”。因此，我们要抓住这一关键环节，用**[分论点]**的突破打开**[主题]**的新局面,。

#### 4. 结尾模板（“实干型”）

- **逻辑**：名言 + 呼吁实干/信心 + 升华愿景。

- **模板**：

  > “纷繁世事多元应，击鼓催征稳驭舟”。**[主题]**的新蓝图已经绘就，新征程即将开启。只要我们撸起袖子加油干，以时不我待、只争朝夕的精神，奋力走好**[主题]**的长征路，就一定能够实现**[主题]**的高质量发展，在实现中华民族伟大复兴的进程中书写新史诗、创造新奇迹。

#### 5. 具体示例说明：以“流动与新生”为例,

- **题目**：参考给定资料，以“流动与新生”为题目，写一篇文章。
- **审题**：
  - 这是一道典型的**双主题（关系型）**作文。
  - 题干提示：“大河奔流...在流动中焕发生机...产生新机遇”。暗示**流动是前提，新生是结果**，二者相辅相成。
- **谋篇布局（建议结构）**：
  - **第一段（开头）**：套用“苟日新”或“时代大潮”模板，引出要统筹流动与新生。
  - **第二段（含义）**：解释“流动”不仅仅是水的流动，更是人才、资源、要素的流动；“新生”是指新机遇、新动能。
  - **第三段（关系）**：分析**流动促进新生**。引用材料中交通路网带动经济发展的例子。
  - **第四段（关系/对策）**：分析**新生倒逼流动**或**如何促进流动**。引用材料中产业升级的例子。
  - **第五段（对策）**：提出具体措施（如打破壁垒、创新机制等）。
  - **第六段（结尾）**：套用“击鼓催征”模板，升华主题。

**总结：** 同学，不要畏惧大作文。**练好字、审好题（定结构）、背好几个核心逻辑模板（如“一子落”、“苟日新”）**，你就掌握了通往高分的钥匙。剩下的就是去练习，把这些模板内化成你自己的语言。加油！

# 标题

你好！我是你的申论大作文专职教师。关于大作文的标题，你一定要高度重视，因为它是阅卷人看到的第一个信息，被称为文章的“眼睛”和“灵魂”。阅卷时间非常短，一个好的标题能瞬间告诉阅卷人：**你没有跑题，你的观点很明确**。

下面我将从核心要义、拟定逻辑、具体指南及模板示例四个方面，为你进行“保姆级”的辅导。

### 一、 拟定标题的核心要义：精准与直白

申论不是文学创作，不需要朦胧美。拟定标题的核心要义只有两条：

1. **涵盖核心关键词（判分点）**：标题必须包含题干要求的**主题词**。阅卷人在看标题时，第一眼找的就是有没有那个“词”。如果有，你就安全了；如果没有，风险很大。
2. **观点明确（亮明态度）**：标题要直接传达你的中心思想。不能含糊其辞，要让阅卷人一眼看出你要写什么、你的态度是什么。例如，写“加强制度建设，根除腐败问题”就比“关于腐败问题的思考”要好得多，因为前者观点鲜明。

**什么样的标题能获认可？**

- **形式上**：简洁凝练，居中书写。如果是主副标题，要排版美观（主标题居中，副标题在主标题右下方或同样居中）。
- **内容上**：切中题干，拒绝文艺风。千万不要起诸如“随风而逝”、“有一群人需要我们爱”这种让阅卷人摸不着头脑的文艺标题，必须直击主题，如“以水为师”、“试谈有与无”。

------

### 二、 标题拟定的逻辑：三步走

不要凭空捏造标题，要遵循以下逻辑：

1. **审题干，找对象**：看题干让你写什么。是围绕“行政执法”，还是“资源型城市转型”，还是“好政策”。
2. **定立意，找角度**：确定你的总论点是什么。是强调“对策”（怎么办），还是强调“意义”（为什么），还是分析“关系”（A与B）。
3. **选结构，套模板**：根据确定的立意，选择是写“单标题”还是“双标题”（主副标题）。

------

### 三、 具体指导与模板（干货）

为了确保考场上快速成题，我为你整理了三类最实用的标题写法。

#### 1. 单标题：简洁有力，万能套用

这种写法最安全，适合绝大多数题目。核心公式是：**动词 + 主题词** 或 **万能句式 + 主题词**。

- **常用模板**：
  - 《做好**[主题词]**这篇大文章》
  - 《谱写**[主题词]**新篇章》
  - 《下好**[主题词]**这盘棋》
  - 《让**[主题词]**大放异彩》
  - 《答好**[主题词]**这道题》
- **适用场景**：当你觉得立意比较单一，或者想不出对仗句时，直接套用。例如主题是“行政执法”，就写《做好行政执法这篇大文章》。

#### 2. 双标题（主副标题）：信息量大，容错率高

这是我**最推荐**的写法。主标题用虚写（宏观/比喻），副标题用实写（具体主题/来源），能完美涵盖所有得分点。

- **核心逻辑**：
  - **主标题**：动词+宏观愿景/比喻（吸引眼球）
  - **副标题**：——由“关键词”想到的 / 对“关键词”的思考（点明题意）
- **常用模板**：
  - 主标题：《**[动词+主题]**新篇章/大文章》
  - 副标题：——由“**[材料/题干核心词]**”想到的
- **具体示例（以“行政执法”为例）**：
  - 主标题：做好行政执法这篇大文章
  - 副标题：——由“力”“理”“利”想到的
  - *解析：主标题展示高度，副标题直接点出题干要求的三个关键要素，阅卷人一看就知道你读懂了题目。*

#### 3. 对仗式标题：工整美观，展现逻辑

这种标题由两个短句组成，读起来朗朗上口，通常体现“对策+目的”或“主题+意义”的关系。

- **逻辑结构**：
  - **对策 + 主题/目的**：如《坚持多措并举，推进环境整治》。
  - **主题 + 意义/愿景**：如《加强环境整治，建设美丽中国》。

------

### 四、 具体示例说明

为了让你更直观地理解，我们拿几个真题或者模拟题来演练一下：

**示例 1：主题为“弘扬黄河精神”**

- **单标题**：《做好弘扬黄河精神这篇大文章》
- **对仗式**：《弘扬黄河精神，凝聚发展力量》

**示例 2：主题为“行政执法中的力、理、利”**

- **双标题（推荐）**：
  - 主：做好行政执法这篇大文章
  - 副：——由“力”“理”“利”想到的
- **变体**：
  - 主：答好行政执法这道题
  - 副：——由“力”“理”“利”想到的

**示例 3：主题为“为民办事（办好事、好办事、事办好）”**

- **双标题**：
  - 主：谱写为民办事新篇章
  - 副：——由办好事、好办事、事办好想到的
- **文艺变体**：
  - 主：为民办事的“三字经”
  - 副：——对办好事、好办事、事办好的思考

**特别提醒（避坑指南）：**

1. **给定标题**：如果题干明确说“以《...》为题”，**绝对不能**自己改标题，也不要加副标题，否则直接扣分。
2. **自拟题目**：如果题干说“以...为话题”或“为主题”，你**必须**自拟标题，不能直接把话题当标题。比如以“人生的奖杯”为话题，标题不能只写《人生的奖杯》，而要写《齐心协力捧起人生奖杯》。
3. **字数与排版**：标题不要太长，尽量控制在一行以内。如果是主副标题，尽量两行写完，保持卷面清爽。

同学，掌握了这些套路，标题这一关你就稳了！现在你可以拿几道真题练练手，把这些模板内化成你自己的本能。

# 审题

你好！作为你的申论大作文专职教师，我非常理解你在审题和谋篇布局上的困惑。其实，申论大作文并不是单纯的文学创作，而是一道**有着客观评分标准的“大主观题”**。

要想拿高分，关键在于像“解数学题”一样去拆解题干，像“拼积木”一样去搭建结构。下面我将手把手教你如何从题干和材料中精准提取总分论点，并完成谋篇布局。

### 一、 核心心法：立意 = 题干（核心）+ 材料（参考）

很多同学容易跑题，是因为过度依赖材料而忽略了题干，或者完全脱离材料空谈。请记住这个公式：**总论点（立意）是根据题干确定的，材料只是为了辅助你理解题干的含义**。

#### 1. 依据题干定“调子”

- **给定标题/话题**：如果题干明确要求“以……为题”或“以……为话题”，**总论点必须直接围绕这个词展开**。
  - *示例*：2019年浙江卷要求以“互联互通”为话题，虽然材料大篇幅讲“运河文化”，但你的总论点必须是写“互联互通”，运河文化只是论证它的素材。如果写成“弘扬运河文化”，就是跑题。
- **给定引言（句子）**：如果题干是“请深入思考这句话……”，**总论点就是对这句话核心含义的解释和延伸**。
  - *示例*：题干引用“夜色难免黑凉，前行必有曙光”，你需要分析出“夜色”代表困难，“曙光”代表希望，“前行”代表行动。总论点就是：面对困难要有信心，要务实行动。
- **给定故事/案例**：如果题干要求从某个故事说开去，**总论点就是这个故事背后的哲理或精神**。
  - *示例*：题干讲“刘女士和李老太的故事”，总论点应提炼为“学好人、办好事、得好报——发挥榜样作用共建精神家园”。

#### 2. 依据材料找“实锤”

- 当题干关键词比较抽象（如“赶考”、“治慧”、“流动与新生”）时，必须**回到材料中去找定义**。
  - *示例*：“以水为师”，你需要去材料里找“水”到底代表什么精神（如以柔克刚、利万物不争、随物赋形等）。材料里有的才是得分点，自己瞎编的不是。

------

### 二、 进阶技巧：如何精准获取分论点（骨架搭建）

分论点是支撑总论点的“柱子”，阅卷人看分论点主要看**段首句**。分论点的来源有三种情况：

#### 1. 题干直接“送分”

有时候题干已经把分论点告诉你了，直接拿来用即可。

- **示例**：题干要求“围绕行政执法中的**‘力’‘理’‘利’**进行思考”。
- **操作**：你的三个分论点就是：
  1. 行政执法要有“力”（力度/公信力）；
  2. 行政执法要讲“理”（规范/说理）；
  3. 行政执法要谋“利”（维护群众利益）,。

#### 2. 题干给“线索”，材料填“血肉”

题干给出了逻辑方向，具体内容要从材料概括。

- **示例**：题干涉及“变与不变”。
- **操作**：这暗示了**关系型**或**辩证型**结构。你需要去材料里找：什么是变？什么是不变？二者关系是什么？分论点通常是：
  1. **含义/辩证关系**：变是时代的呼唤，不变是初心的坚守；
  2. **对策A**：顺应时代之“变”（创新机制、更新技术）；
  3. **对策B**：坚守法治之“不变”（公平正义、为民初心）。

#### 3. 题干无“暗示”，全靠材料“挖”

题干只说“结合给定资料写一篇文章”，没有任何关键词提示。

- **操作**：这是对全篇材料的**归纳概括**。你需要梳理材料的逻辑：
  - 如果材料都在讲问题，分论点就是**分析问题+解决对策**。
  - 如果材料都在讲意义，分论点就是**意义1+意义2+意义3**。
  - **高频词法**：材料中反复出现的对策词、高档词（如创新、协调、绿色），往往就是分论点。

------

### 三、 实战指南：如何谋篇布局（结构选型）

申论作文没有绝对固定的模板，**结构取决于题干要求**。千万不要不管什么题都硬套“三个对策”或“三个意义”。

#### 1. 关系型结构（A与B）

- **适用场景**：题干出现两个关键词（如“流动与新生”、“有与无”、“红线与红利”）。
- **布局指南**：
  - **开头**：引出A与B。
  - **第二段（核心判分点）**：**专门分析A与B的辩证关系**（如相辅相成、互为支撑）。这部分没写会被扣大分,。
  - **第三段**：A对B的作用/如何做好A。
  - **第四段**：B对A的作用/如何做好B。
  - **第五段**：统筹兼顾的对策。
  - **结尾**：升华。

#### 2. 综合分析型结构（是什么-为什么-怎么办）

- **适用场景**：题干是抽象话题（如“赶考”、“治慧”）或引言。
- **布局指南**：
  - **开头**：引用名言/点题。
  - **含义段**：解释关键词的深层含义（这是基础）,。
  - **意义段/原因段**：分析为什么要这么做（这是关键）。
  - **对策段**：具体怎么做（这是根本）。
  - **结尾**：总结。

#### 3. 策论文结构（问题-对策）

- **适用场景**：题干明确要求解决问题，或材料全是负面案例。
- **布局指南**：
  - **开头**：概括问题，亮明总论点。
  - **分析段**：简要分析问题的原因或危害（过渡作用）。
  - **分论点1**：对策一。
  - **分论点2**：对策二。
  - **分论点3**：对策三。
  - **结尾**：展望未来。

#### 4. 驳论文结构

- **适用场景**：题干明确要求反驳某种错误观点（如“人文是精彩的，科学是呆板的”）。
- **布局指南**：
  - **开头**：摆出错误观点。
  - **驳论段**：分析该观点错在哪里，提出正确观点（立论）。
  - **对策段**：如何落实正确观点。
  - **结尾**：总结。

### 四、 教师寄语：分论点优化“三板斧”

为了让你的分论点段首句更出彩，建议采用**“虚实结合”**或**“关键词+解释”**的写法：

1. **关键词法**：把核心词前置，后面跟解释。
   - *示例*：“完善的人才选拔制度是实施人才强国战略的基础。”
2. **对仗法**：
   - *示例*：“聚焦两个意识是赶考的基础……；聚焦两个精神是赶考的关键……；聚焦两个品格是赶考的根本……”。
3. **模板逻辑法**（用于扩充内容）：
   - **“一子落而满盘活”**：分论点是关键一子，做好了就能盘活全局。
   - **“木桶效应”**：分论点是短板，必须补齐。

**总结**：审题是方向，材料是宝库。先看题干定“是什么结构”，再看材料填“什么内容”，最后用规范的段首句“亮明观点”。按这个步骤走，你的作文就是一类文的骨架！加油！

# 开头逻辑

你好！我是你的申论大作文专职教师。关于大作文的“凤头”（开头），很多同学都有畏难情绪，要么写得太慢浪费时间，要么写得太偏离题万里。

阅卷人在看开头时，通常只用**几秒钟**扫描核心信息。因此，一个好的开头必须做到**“短、快、美”**：字数控制在150-200字（短），迅速切入题干关键词（快），语言有亮点（美）。

下面我将从写作逻辑、规划方法和评价标准三个维度，为你条分缕析，并给出直接可用的“积木块”。

### 一、 开头的写作逻辑是什么？

申论开头的底层逻辑其实非常简单，就是一个**“三段论”推导过程**。无论你用什么修辞，核心骨架通常只有两种：

#### 1. 转折递进逻辑（“好+坏+总论点”）

这是最经典、最稳妥的逻辑。

- **逻辑链条**：先承认成绩/背景（好） $\rightarrow$ `然而`指出当前存在的问题/挑战（坏） $\rightarrow$ `因此`引出总论点（对策/立意）。
- **适用场景**：绝大多数社会问题类、改革类话题。
- *源理解析*：这种逻辑符合辩证法，先肯定后否定再超越，符合机关公文的思维习惯。

#### 2. 引用引申逻辑（“名言+本质+总论点”）

这是提升文采、震慑阅卷人的逻辑。

- **逻辑链条**：引用一句名言/题干引言 $\rightarrow$ 解释这句话揭示的深刻道理/本质 $\rightarrow$ 联系实际引出总论点。
- **适用场景**：题干给出一句抽象引言，或考察某种精神、理念（如“赶考”、“治慧”）。
- *源理解析*：利用“大道至简”、“理一分殊”等万能句式，将具体话题上升到哲学高度，再落地到总论点。

------

### 二、 如何根据题干和材料规划开头？

不要进考场现编，要根据**题干的暗示**和**材料的特征**，像“拼积木”一样组装开头。

#### 第一步：审题干，定“调子”

看题干是哪种类型，决定用哪套逻辑：

1. **引言式（题干有一句话）**：直接套用**“大道至简”**或**“言近旨远”**模板。先抄写/引用题干那句话，然后说它揭示了什么道理，最后引出论点。
2. **抽象话题式（如“变”、“赶考”）**：套用**“苟日新”**（强调变化）或**“名言/比喻”**模板。强调时代变了，认识也要变，引出新论点。
3. **社会问题/现象式**：套用**“转折递进”**或**“问题是时代的口号”**模板。先说现状，再说问题，最后提对策。
4. **关系型（A与B）**：套用**“车之两轮、鸟之双翼”**模板，强调二者缺一不可，引出统筹兼顾的论点。

#### 第二步：看材料，填“血肉”

逻辑框架搭好后，中间的“空”填什么？

- **填背景/成绩**：去材料里找第一段或结尾段的宏观背景词（如“新时代”、“转型期”）。
- **填问题/挑战**：去材料里概括核心问题（如“资源枯竭”、“道德失范”）。
- **填案例**：如果材料里有典型案例（如“S市治理”、“某某人物”），可以用一句话高度浓缩，作为引出论点的铺垫。

------

### 三、 怎样评价大作文开头是否切题？

我在阅卷或批改作业时，评价一个开头是否合格，主要看这三点：

1. **含“金”量（关键词）**：开头必须包含题干中的**核心关键词**。如果题干让你写“以水为师”，你的开头里必须出现“水”和“师/学习”；如果题干是“行政执法中的力理利”，开头必须点出这三个字。**没有关键词，直接判为跑题**。
2. **亮“剑”速（总论点）**：总论点必须在**第一段的最后一句**（最晚第二段开头）明确亮出。不要含蓄，不要让阅卷人猜，要用“因此，我们应……”这样的句式直接给出来。
3. **逻辑“顺”（连接词）**：看是否有清晰的起承转合。比如有没有用“然而”、“但是”来转折，有没有用“古人云”来引经据典。逻辑顺畅，阅卷人读起来才舒服。

------

### 四、 教师手把手：实战模板与示例

为了让你能直接上手，我为你准备了三套最实用的“万能”开头模板。

#### 1. “苟日新”模板（适用于创新、变化、理念类）

- **模板**：

  > 古人云：“苟日新，日日新，又日新”。时代在发展，我们对于**[主题词]**的认识也在不断变化。从过去的**[旧观念/旧现状]**到现在的**[新观念/新挑战]**，我们越发认识到**[题干引言/核心观点]**的重要性。因此，我们应顺应时代潮流，做好**[总论点]**这篇大文章。

- **示例（主题：提升想象力）**：

  > 古人云：“苟日新，日日新，又日新”。随着时代发展，我们对于“想象力”的认识也在不断变化。尤其是随着“想象力经济”时代的到来，我们越发认识到提升想象力的重要性。所以我们应该因时而变，做好提升想象力这篇大文章。

#### 2. “大道至简”模板（适用于题干有引言/抽象道理）

- **模板**：

  > 古人云：“大道至简”。**[题干引言或关键词]\**看似简单，却蕴含着深刻的哲理。它不仅揭示了\**[主题]\**的本质规律，更启示我们在新时代要\**[结合材料的启示]**。深入思考这句话，对于我们做好**[总论点]**具有重要的指导意义。

- **示例（主题：以水为师）**：

  > 古人云：“大道至简”。老子“上善若水”的观点看似简单，却包含了深刻的哲理。它启示我们，水柔而能克刚、静而能映物、动而能变化。深入思考水的智慧，对于我们提升智慧与情操、治理城市水系具有重要的指导意义。

#### 3. “问题导向”模板（适用于解决问题、民生类）

- **模板**：

  > 马克思曾说：“问题是时代的口号”。当前，我们在**[领域]**取得了显著成绩，但也要清醒地看到，**[问题关键词]**等问题依然突出。这些问题倒逼我们必须**[总论点]**。只有牵住**[核心对策]**这个“牛鼻子”，才能谱写**[主题]**的新篇章。

- **示例（主题：行政执法）**：

  > 马克思曾说：“问题是时代的口号”。当前，我国法治建设成效显著，但在行政执法领域，仍存在“一刀切”、暴力执法等问题。这些问题倒逼我们必须反思执法的初衷。因此，我们必须统筹好“力”、“理”、“利”的关系，做好行政执法这篇大文章。

**总结：** 同学，开头的核心任务就是**“引出总论点”**。考试时不要追求标新立异，只需根据题干类型，从我给你的这三个模板里挑一个最合适的，把材料关键词填进去。**练好字，写工整，逻辑顺，关键词在**，这就是一个满分的一类文开头！
